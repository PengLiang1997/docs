# 基本信息

## 一、更改头像
点击左下角用户按钮，点击用户信息按钮，点击基本信息tab页，进入用户基本信息维护页面，点击**更改头像**按钮，弹出文件选择框
![更改头像](/pic/userinformation/baseinformation/headportrait1.jpg)
![更改头像](/pic/userinformation/baseinformation/headportrait2.jpg)

选择图片文件后，点击确定

## 二、维护基本信息
输入用户昵称后，在用户基本维护页面选择要输入的信息输入后，点击底部**保存**按钮

*用户昵称为必填项*
![维护基本信息](/pic/userinformation/baseinformation/baseinformation.jpg)
